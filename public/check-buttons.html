RWA Hackathon - 按鈕檢查工具

或者手動檢查：

  1. 訪問 https://hackathon.com.tw/
  2. 按 F12 打開開發者工具
  3. 在 Console 輸入以下代碼： 
    const links = document.querySelectorAll('a');
const forum = Array.from(links).find(a => a.href && a.href.includes('forum2025'));
console.log('Forum button:', forum);
console.log('Text:', forum ? forum.textContent : 'NOT FOUND');
  4. 查看結果
