🔄 清除緩存工具

修復 "Add New Scan" 錯誤

如果您看到 "Unexpected error" 訊息，請按照以下步驟操作：

方法 1：自動清除（推薦）

  1. 點擊下方的「清除緩存並重新載入」按鈕
  2. 等待頁面自動跳轉到掃描管理頁面
  3. 重新嘗試添加掃描類型
✅ 緩存已清除！正在跳轉...

方法 2：手動清除（如果方法 1 無效）

  1. Chrome/Edge：
    • 按 Ctrl + Shift + Delete (Windows) 或 Cmd + Shift + Delete (Mac)
    • 選擇「緩存的圖片和文件」
    • 點擊「清除數據」
  2. Firefox：
    • 按 Ctrl + Shift + Delete
    • 選擇「緩存」
    • 點擊「立即清除」
  3. 重新載入頁面（F5 或 Ctrl + R）

⚠️ 注意事項

1. 確保您已經登入並有管理員權限

2. 如果問題持續，請嘗試使用無痕模式（Incognito/Private）

3. 確認您的 Firebase token 沒有過期（重新登入）
