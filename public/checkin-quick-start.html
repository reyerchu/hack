💡 使用說明

管理員操作：

  • 點擊「開始掃描報到」進入掃描頁面
  • 允許瀏覽器使用攝影機
  • 將參賽者 QR code 對準掃描框
  • 系統自動記錄報到

👥 參賽者 QR Code

參賽者可在個人資料頁面找到自己的 QR code

路徑：登入 → 個人中心 → 顯示於頁面頂部

⚠️ 權限要求

需要以下任一權限才能使用此系統：

  • admin（管理員）
  • super_admin（超級管理員）
  • organizer（主辦方）
